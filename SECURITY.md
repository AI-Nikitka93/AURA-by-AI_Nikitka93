# Security Policy

AURA is a public portfolio site. Please report vulnerabilities, privacy issues, or data-handling concerns privately when possible.

## Supported scope

- The current `main` branch.
- The latest published GitHub Pages deployment.

## How to report

- Use GitHub's private security advisory flow for sensitive reports.
- Do not include exploit details, credentials, or personal data in public issues.
- If a private advisory is unavailable, open a public issue with only a high-level summary and mark it as security-related.

## Response expectations

This project is maintained on a best-effort basis. There is no guaranteed SLA.
