## 🔁 ROLLBACK READINESS
- GitHub Pages теоретически поддерживает deployment history через GitHub Actions, а Cloudflare Workers поддерживает `wrangler rollback`, но в самом репозитории rollback procedure не описана и не проверена на текущем контуре.
