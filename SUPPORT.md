# Support

## What This Repository Supports

AURA is a public portfolio showcase. Support is best-effort and focused on:
- broken local setup instructions
- packaging or documentation mistakes
- public demo regressions on the primary Vercel URL or the GitHub Pages mirror
- clearly scoped bugs in the React frontend or Worker integration path

## Best Support Paths

- Bug or regression: open a GitHub issue with reproduction steps
- Security or privacy concern: use the private route described in [SECURITY.md](SECURITY.md)
- Documentation or packaging problem: open a GitHub issue or pull request

## What This Repository Does Not Offer

- product SLA or guaranteed response times
- custom implementation support
- licensing approval for reuse of the code, copy, or design assets

## Before Opening An Issue

1. Check the canonical docs in [README.md](README.md) and [RELEASE_RUNBOOK.md](RELEASE_RUNBOOK.md).
2. Confirm whether the problem is on the Vercel frontend, the GitHub Pages mirror, or the Worker/API path.
3. Include screenshots for visual bugs or route mismatches.
