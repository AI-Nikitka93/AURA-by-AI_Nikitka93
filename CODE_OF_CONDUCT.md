# Code of Conduct

## Our Standard

This repository is a public portfolio surface. Discussions, issues, and pull requests should stay respectful, specific, and useful to other readers.

Examples of expected behavior:
- giving actionable technical feedback
- disagreeing without personal attacks
- keeping bug reports reproducible and concise
- respecting privacy, licensing, and disclosure boundaries

Examples of unacceptable behavior:
- harassment, insults, or intimidation
- publishing private data, credentials, or exploit details in public threads
- spam, trolling, or deliberately disruptive comments
- demanding support as if this repository had a commercial SLA

## Scope

This policy applies to repository issues, pull requests, code review comments, and other public interactions tied to this repository.

## Enforcement

The maintainer may edit, hide, lock, or remove interactions that violate this policy, and may close contributions that do not fit the repository’s scope.

## Reporting

For behavior concerns, open a private maintainer contact on GitHub if possible. For security or privacy concerns, follow [SECURITY.md](SECURITY.md).
